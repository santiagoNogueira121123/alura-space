# 🌌 Alura Space

## Este projeto foi desenvolvido como parte da trilha de especialização em Django da plataforma Alura. A aplicação utiliza o framework Django para gerenciar o back-end, roteamento e exibição de conteúdos dinâmicos.

# 🚀 Sobre o Projeto

##  O Alura Space é uma galeria interativa de fotografias espaciais. O objetivo principal é oferecer uma experiência de exploração onde o usuário pode visualizar imagens do cosmos de forma organizada.
